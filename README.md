--- 
### Github-Insights [Render Link](https://github-insights-pxwk.onrender.com/)

#### Summary
Input your github user or another user's link into bar, press fetch for a run down of all their github stats for a given year.

---
#### App Installation
```bash
# Run these commands:

# RECOMMENDED for local test
docker compose up --build

# OR RUN LOCALLY (VERY TEDIOUS)
pip install -r requirements.txt
cd./backend
python main.py

cd./frontend
npm run dev
```
---

#### Deployment Ready Code
```bash
# Deployment ready using docker e.g somthing like render:

# add environment variables in render
TOKEN=YOURGITTOKEN
SERVER1=renderserverlink
```
